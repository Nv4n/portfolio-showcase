# Portfolio Showcase

This is a Next.js project showcasing a portfolio of projects. It includes various components such as navigation menus, cards, forms, and more. The project is built using Next.js, React, and Tailwind CSS.

## Features

- Navigation menu with links to different sections of the portfolio
- Cards to display project information
- Forms for signing up, signing in, and resetting passwords
- Image uploader dialog
- Account settings form

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository: `git clone <repository-url>`
2. Install dependencies: `npm install` or `pnpm install`
3. Start the development server: `npm run dev` or `pnpm dev`
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Technologies Used

- Next.js
- React
- Tailwind CSS
- React Hook Form
- Lucide React icons
- Radix UI
